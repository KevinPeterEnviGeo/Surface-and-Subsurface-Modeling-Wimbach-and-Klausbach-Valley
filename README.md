# Surface-and-Subsurface-Modeling-Wimbach-and-Klausbach-Valley

Within a course we explored the surface and the subsurface of the Wimbach and Klausbach valley in Berchtesgaden National Park. The varying methods lead to diverse results which are discussed in detail in the report/paper.



\# Surface-and-Subsurface-Modeling-Wimbach-and-Klausbach-Valley



Ober- und Untergrundmodelle



University of Bayreuth (2023)



\## Overview



This repository contains only a slimmer of the data handled within this project. Unfortunately, the GIS and TLS data (where most of the work was done) is too big to share via GitHub, if there is interest I would be keen to share the data with you through another platform. The ERT data on the other hand was not entirely collected by myself. therefore I wont share it here. 



This was an in-depth exploration of the two main valleys in a geomorphologic way.
